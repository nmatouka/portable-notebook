Open marimo notebooks — live, offline, by double-click.

    Open a notebook

    Drag a .mnote or .py onto this window, or use File ▸ Open Notebook (⌘O).

  • ⇪
    Share one

    Open a notebook, then File ▸ Export as .mnote bundles it so it opens offline anywhere Carrel is installed.

  • ★
    New here?

    File ▸ Open Example runs a live notebook so you can see how it feels.

A .mnote is just a marimo notebook; Carrel supplies the runtime, so nothing needs installing to view one.
